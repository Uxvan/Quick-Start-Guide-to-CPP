# Quick-Start-Guide-to-C-
利用例子快速掌握cpp

1. **基础语法过渡**:变量类型、编译运行流程(g++怎么用)、输入输出、条件循环、函数
2. **指针与内存**:栈vs堆、指针和引用的区别、new/delete、悬空指针
3. **数组与字符串**:C风格数组 vs std::vector、std::string操作
4. **面向对象**:类、构造析构函数、继承、虚函数/多态
5. **模板与泛型**:function template、class template
6. **STL标准库**:vector/map/set等容器,iterator,algorithm库常用函数
7. **内存管理进阶**:RAII、智能指针(unique_ptr/shared_ptr)——现代C++写法,替代手动new/delete
8. **编译与工程**:头文件/源文件分离、Makefile/CMake基础
